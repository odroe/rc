# RC v0.3.3

To install RC v0.3.3 with command:

```bash
dart pub add orm:^0.3.3
```

Or update your `pubspec.yaml` file:

```yaml
dependencies:
  rc: ^0.3.3
```

## What's Changed

* Optimize warning output
